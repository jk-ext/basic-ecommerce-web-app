# Basic E-Commerce Web Application

This project is a simple e-commerce style web application built using PHP, MySQL, HTML, and CSS.

## Features
- User registration and login
- Order placement and tracking
- Database-backed backend
- Basic frontend interface

## Technologies Used
- HTML
- CSS
- PHP
- MySQL

## Database Setup
- Import the `setup_database.sql` file into MySQL
- Configure database connection in PHP files

## Purpose
This project was developed as an academic mini project to understand backend development, database relationships, and web application flow.
